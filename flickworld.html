FlickWorld Role Playing Game

This is my first ever 2D game. FlickWorld is a role-playing game with interactive NPCs and a basic combat system. Players can explore a rich, pixelated world full of adventure and challenges.

  • Engine: Unity
  • Language: C#
  • Development Time: 3 months
  • Key Features: Interactive NPCs, Basic Combat System, Inventory Management
